# Dream Gold — проектный CLAUDE.md

Авторский ювелирный бренд. Премиум сайт-витрина с личным кабинетом. Цель — уровень Awwwards.

## Что это за сайт
- Бренд **Dream Gold** — авторские ювелирные изделия ручной работы, собственное производство в Украине.
- Модель: **витрина + заявка** (inquiry). БЕЗ онлайн-оплаты и БЕЗ цен на сайте. CTA везде — «Залишити заявку».
- Язык контента — **только украинский** (`lang="uk"`). Обращение к клиенту — на **«ви»**.
- Личный кабинет: профиль, мои заявки, збережене (wishlist).

## ⚠️ Позиционирование — жёсткое правило
Публичный сайт = легитимный бренд авторских изделий. **НЕЛЬЗЯ** упоминать/рекламировать копии чужих брендов, их названия и торговые марки, слово «копия», показывать чужие логотипы. Никаких цен на изделия. Подача: «втілимо будь-яку ідею», вдохновлённые иконами стиля — но как изделие Dream Gold.

## Стек
- Next.js 16 (App Router) + React 19 + TypeScript (strict)
- Tailwind v4 + shadcn/ui (radix, preset nova) + Lucide
- Анимация: GSAP (+ @gsap/react), Lenis (smooth scroll)
- 3D точечно: react-three-fiber + drei + postprocessing
- Backend/CMS: Payload v3 (каталог + заявки + кабинет), Postgres на Supabase  ← настраивается
- Деплой: Vercel (позже)

## Команды
- `npm run dev` — дев-сервер (Turbopack), http://localhost:3000
- `npm run build` — прод-сборка · `npm run lint` — ESLint · `npx tsc --noEmit` — типы

## Дизайн-система (app/globals.css)
- Токены-цвета: `--ivory #F6F0E6`, `--cream`, `--sand`, `--ink #1A1714`, `--ink-muted`, `--ink-faint`, `--line`, `--gold #C2A36B`, `--gold-deep`, `--gold-soft`. Tailwind: `bg-ivory`, `text-ink`, `text-gold`, `border-line`…
- Шрифты: **Playfair Display** (`font-display`, заголовки) + **Manrope** (`font-sans`, текст). Обе с кириллицей — НЕ заменять на шрифты без кириллицы.
- Типошкала: `text-display/h1/h2/h3/eyebrow`. Скругления минимальные (`--radius: .25rem`).
- Motion: `--ease-out`, `--ease-inout`, `--d-*`, `--lenis-lerp`. Тёмные секции — обёртка `.dark` (тёплый графит).

## Структура (multi-page)
Меню: Каталог · Індивідуальне · Про нас · Шоурум · Контакти (+ іконки Кабінет/Збережене). Архитектура — **короткая главная-хаб + отдельные роуты**; глубина (каталог, майстерність, история) живёт в отдельных вкладках, НЕ на главной. Главная (намеренно короткая): прелоадер → hero → манифест → Розділи (3 двери) → вибрані роботи → заявка → footer. Роуты: `/catalog` (+`/[slug]`), `/individual`, `/about`, `/showroom`, `/contacts`, `/account`, `/wishlist`.

## Конвенции
- TypeScript strict, без `any` (вынужденно — комментарий почему).
- A11y; `prefers-reduced-motion` на всех тяжёлых анимациях; мобайл — приоритет. Цели: LCP<1.5s, CLS<0.05, INP<100ms, 60fps.
- Тексты пишет заказчик (профессиональный украинский) — оставляем понятные слоты под контент.
- Секреты только в `.env*` (не коммитим); серверная авторизация — проверкой пользователя на сервере.
- Код — в стиле окружения.

## Чего не трогать
- `.env*`, секреты, сгенерированные типы.
- Реальный логотип в `public/brand/` — не перерисовывать (только векторизация/перекрас под палитру).
